\*\*一款专为小型生产场景设计的实时缺陷检测工具\*\*  

无需工业相机、无需编程基础、无需高额成本，普通 USB 摄像头即可实现产品表面缺陷的自动检测、数据记录与日报生成。



---



\## ✨ 核心亮点



\- ✅ \*\*低成本入门\*\*：普通摄像头即可，无需工业设备，100MB 存储就能跑。

\- ✅ \*\*零编程上手\*\*：提供独立 exe 文件，双击运行，Windows 10/11 均可使用。

\- ✅ \*\*全流程覆盖\*\*：实时检测 → 样本采集 → 日报生成 → 数据清洗，一套搞定。

\- ✅ \*\*开源可定制\*\*：完整 Python 源码开放，支持二次开发与算法调优。

\- ✅ \*\*数据化赋能\*\*：自动记录质检数据，生成 Excel 日报和趋势图，质量优化有据可依。

\## 🚀 快速开始



\### 1. 下载与准备

\- 从 \[Releases](https://github.com/Tidal-wave-First/eagle-eye/releases) 下载最新版压缩包（包含 exe 文件和示例模板）。

\- 准备一张\*\*无缺陷的良品图片\*\*，重命名为 `template.jpg`，放入软件所在文件夹。



\### 2. 运行实时检测

\- 双击 `鹰眼实时检测系统.exe`。

\- 将产品对准摄像头，系统自动检测（约每秒1次），缺陷位置用红框标记。

\- 按 \*\*Q 键\*\* 退出（勿点关闭按钮）。



\### 3. 查看日报

\- 检测过程中自动生成 `鹰眼记录.csv`、`缺陷日报.xlsx`、`缺陷趋势图.png`。

\- 日报按日期汇总缺陷数，趋势图直观显示质量波动。



\### 4. 采集样本（可选）

\- 双击 `图片捕捉工具.exe`，按 \*\*空格键\*\* 保存当前画面至 `captures` 文件夹。



\### 5. 数据清洗（可选）

\- 运行 `闪电数驿.exe`，可对 CSV/Excel 进行排序、清理、统计。

\## 🔧 技术原理



\- \*\*检测算法\*\*：采用差影法，将当前帧与模板图像对齐后做像素差异，通过阈值分割识别缺陷区域（面积 >200 像素）。

\- \*\*图像对齐\*\*：基于 ORB 特征点匹配，可容忍一定程度的旋转和缩放。

\- \*\*数据存储\*\*：缺陷记录存入 CSV，日报自动生成 Excel 并设置日期格式。

\## ❓ 常见问题



\*\*Q：为什么点击关闭按钮退不出？\*\*  

A：OpenCV 窗口机制限制，请务必按 \*\*Q 键\*\* 退出。



\*\*Q：检测结果不准怎么办？\*\*  

A：检查模板图片是否清晰、光线均匀；摄像头是否固定；产品表面是否有反光/阴影。可调整源码中的面积阈值。



\*\*Q：没有摄像头可以测试吗？\*\*  

A：exe 版本必须连接摄像头。如需图片测试，可运行 Python 源码中的 `defect\_demo.py`。



\*\*Q：Excel 日期显示为数字？\*\*  

A：系统已自动设置日期格式，若仍显示数字，请手动选中日期列 → 右键 → 设置单元格格式 → 日期。



\*\*Q：报错“无法打开摄像头”？\*\*  

A：确认摄像头未被其他程序占用（如微信、QQ），插拔摄像头后重试。
